# 模板库

项目模板、文档模板、配置文件模板等。

## 建议模板

- `project-node/` - Node.js 项目模板
- `project-python/` - Python 项目模板
- `project-nextjs/` - Next.js 项目模板
- `docker-compose/` - Docker Compose 模板
- `github-actions/` - GitHub Actions 工作流
- `prd/` - 产品需求文档模板
- `api-spec/` - API 规范模板

## 使用方式

```bash
# 复制模板到新项目
cp -r ~/ai-agents-skills/templates/project-node ./my-project
```
