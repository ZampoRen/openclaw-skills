## 二、标题规范

### 要求

| 要素 | 要求 |
|------|------|
| 长度 | 15-25 字 |
| 关键词 | 必须包含核心技术词 |
| 钩子 | 痛点/收益/好奇心 |
| 数字 | 有具体数字更佳 |

### 示例

| ❌ 差标题 | ✅ 好标题 |
|----------|----------|
| "Qwen3.5 部署教程" | "Qwen3.5 本地部署完全指南：8GB 显存也能跑，速度提升 3 倍" |

| "React Hooks 讲解" | "React Hooks 深度解析：这 5 个陷阱 90% 的人都踩过" |

### 标题公式

```
[技术主题] + [核心价值/痛点] + [数字/结果]

示例：
- "Qwen3.5 本地部署" + "8GB 显存也能跑" + "速度提升 3 倍"
- "Docker 入门" + "30 分钟搭建" + "第一个容器应用"

## 六、文生图提示词规范

### 结构

```
[画面主体描述]，[风格关键词]，[色彩基调]，[构图方式]，[细节要求]，[技术参数]
```

### 必备元素

| 元素 | 说明 | 示例 |
|------|------|------|
| 画面主体 | 具体物体/场景 | "一台打开的笔记本电脑" |
| 风格描述 | 现代/极简/科技感等 | "科技感十足，现代简洁" |
| 色彩基调 | 冷色/暖色/渐变 | "深蓝色渐变，带粒子效果" |
| 构图方式 | 中心/三分/对称 | "中心构图，主体突出" |
| 细节要求 | 高清/细节丰富等 | "高清，细节丰富" |
| 技术参数 | --ar --v --style 等 | "--ar 16:9 --v 6.0 --style raw" |

### 完整示例（微信公众号头条封面）

```
一张科技感十足的插画，主题是"本地 AI 部署"。画面中央是一台打开的笔记本电脑，
屏幕中浮现出一个发光的神经网络结构，神经网络由蓝色和紫色的光点组成，呈现
出立体的 3D 效果。电脑周围环绕着几个漂浮的芯片图标和数据流线条，背景是深
蓝色渐变，带有轻微的粒子效果。整体风格现代、简洁、专业，适合科技类公众
号文章封面。**核心内容集中在画面中央**，确保裁剪后依然完整。画面比例 2.35:1，
高清，细节丰富。--ar 2.35:1 --v 6.0 --style raw
```

**注意：** 微信公众号头条封面是 **2.35:1**（900×383 像素），不是 16:9！
- 核心内容要放在画面中央 383×383 区域
- 列表页只显示中间正方形区域
- 点进文章后才显示完整长图
