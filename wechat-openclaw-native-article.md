# 微信原生 OpenClaw 接入，一篇看懂

> 微信 8.0.69+ 可以直接接入 OpenClaw 了。
>
> 一句话：**能用的东西，但别指望太多。**

---

## 怎么装？

就一条命令：

```bash
npx -y @tencent-weixin/openclaw-weixin-cli@latest install
```

没了。

执行完会弹个二维码，扫一下，绑定成功。

然后在微信通讯录里找"微信 ClawBot"，发消息就能用。

---

## 注意事项

**安卓：** 微信 8.0.69+ 直接装，能用

**iOS：** 微信 8.0.70 可以使用，注意更新版本，如果版本低扫码会提示你更新

**鸿蒙：** 不支持

**电脑端：** 不显示，只能手机端用

---

## QClaw vs 微信原生，怎么选？

直接上对比：

| | QClaw | 微信原生 |
|---|---|---|
| 安装 | 一键 | 命令行 |
| 模型 | 腾讯内置（就那样） | 自己配（想用什么用什么） |
| 外网 | 不能 | 能 |
| 技能 | 5000+ 预置 | 自己装 |
| 免费额度 | 4 千万 token/天 | 自理 |
| 适合谁 | 不想折腾的 | 爱折腾的 |

---

## 我的结论

**QClaw 的问题：**
- 默认模型不够聪明 （可以配置使用自己的模型）
- 没法访问外网（Google、GitHub 都别想）
- 强制必须使用自家的技能，无法关闭。

**微信原生的问题：**
- 只能手机端用

**怎么选？**

- 普通用户、图省事 → QClaw
- 要用 GPT/Claude、要访问外网 → 微信原生
- 我？两个都用，不冲突

微信原生 OpenClaw 是个好东西，但现在还在早期。
