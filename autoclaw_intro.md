## 什么是"龙虾"？

"龙虾"是开源 AI 智能体 **OpenClaw** 的昵称。为什么叫龙虾？因为它的图标是一只虾，而且你需要像养宠物一样"喂养"它数据，让它越来越懂你。

之前想养一只"龙虾"，你得：
- 会装 Node.js
- 会敲命令行
- 会配置各种 API
- 折腾半天还不一定能跑起来

**现在，智谱推出了 AutoClaw（中文名"澳龙"）—— 一键安装版 OpenClaw。**

## 有多简单？

### 之前装 OpenClaw：
```bash
# 1. 装 Node.js
# 2. 克隆仓库
git clone https://github.com/openclaw/openclaw.git
# 3. 安装依赖
npm install
# 4. 配置环境变量
# 5. 配置模型 API
# 6. 启动服务
npm start
# 7. 祈祷别报错😅
```

2. 点击下载

## AutoClaw 有什么本事？

### 🎯 开箱即用
预置了 **50+ 热门技能**，装好就能用：
- 写文章、写邮件
- 查资料、做总结
- 写代码、改 Bug
- 做表格、处理文档
- 订机票、查天气
- ...

不用你自己配搜索 API、生图 API，全都搞定了。

### 🌐 接入飞书/钉钉
可以在聊天框里直接给"龙虾"派活：
> "@龙虾 帮我总结一下这个文档"

它会自动处理，然后把结果发回给你。

### 🧠 支持多种模型
- DeepSeek
- Kimi
- MiniMax
- GLM（智谱自家模型）
- ...

想换模型？改个配置就行。

### 🦞 专属优化模型
智谱专门为 OpenClaw 场景优化了一个模型叫 **Pony-Alpha-2**（内测代号），在工具调用和任务执行上更强。

### 🌍 浏览器自动化
集成了 AutoGLM Browser-Use 能力，"龙虾"可以：
- 自动打开网页
- 填表、点击
- 跨页面操作
- 帮你完成多步骤任务

比如："帮我在携程订一张明天北京到上海的机票"——它真的可以一步步操作。
