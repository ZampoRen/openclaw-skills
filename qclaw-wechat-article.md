### 方式 2：OpenClaw 开源版（适合折腾党）

**适合人群：** 开发者、想自定义配置、需要更多控制权

**安装步骤：**

```bash
# 1. 克隆仓库
git clone https://github.com/openclaw/openclaw.git
cd openclaw

# 2. 安装依赖（需要 Node.js 18+）
npm install

# 3. 配置环境变量
cp .env.example .env
# 编辑.env 文件，填入你的 API Key

# 4. 启动服务
npm run start
```

**配置 Skills：**
```bash
# 查看可用技能
openclaw skills list

# 安装技能
openclaw skills install <skill-name>
```
